================================
Pour le personnel de maintenance
================================

Die meisten Anpassungen, die in den Codebeispielen in diesem Beispiel enthalten sind, sind standardmäßig aktiviert. Wundern
Sie sich also nicht, wenn Sie hier und da seltsame Dinge in Ihrem TYPO3 Backend finden.

Wann immer es möglich ist, wird die Verwendung eines bestimmten Teils des Codes im Quellcode mit der folgenden Art von
PHP-Kommentaren beschrieben

.. code-block:: php

   // USAGE: Core APIs > TYPO3 API overview > Various examples > Support for custom tables in the Page module

Dadurch wird es einfacher, ein bestimmtes Codebeispiel mit einer bestimmten Stelle in einem Handbuch zu verknüpfen.
Umgekehrt sollte es möglich sein, im Quellcode nach einem Kapiteltitel zu suchen und das entsprechende Codebeispiel
zu finden.
